# About RFPS MAPS

This repository is used for generating static parts of 3D scene maps used in the *RFPS* project.
Blender version 3.0.1 and Python 3.10 were used in development.

## How to run

- **Run tests** : `make test`
- **Render model in console** : `make run FILE=Metro.py`
- **Show model in Blender** : `make show FILE=Metro.py` (replace the filepath with the desired script file name)

## Documentation

- **Create documentation** : `make doc`
- **Count lines of code** : `make cloc`
